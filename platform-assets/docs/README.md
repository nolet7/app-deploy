# IDP Overview

## Goal
Build an enterprise-style Internal Developer Platform that can:
- scaffold different application types
- standardize CI/CD
- enforce security controls
- deploy through GitOps
- attach observability and SRE defaults

## Main Modules
1. Developer Portal
2. Template Catalog
3. Bolt Prompt / Code Generation Layer
4. Repo and CI/CD Automation
5. Deployment Automation
6. Secrets and Security
7. Observability and SRE Pack

## Supported App Types (MVP)
- React frontend
- FastAPI backend
- React + FastAPI full-stack
- AI inference API
